# 🏥 小型医院医疗管理系统（C语言控制台版）

> **吉林大学 · 软件工程专业课 · 大一程序设计作业**

一个模拟真实医院门诊与住院流程的**控制台管理系统**，使用 C 语言 + 链表实现，包含患者、医生、护士、管理员四大角色，涵盖挂号、分诊、开药、检查/检验、住院、出院、药品管理、费用统计等核心功能。

---

## ✨ 主要功能

### 病人端
- 患者注册 / 登录
- 现场挂号 / 预约挂号（自动分配排队号）
- 退号、查看排队叫号状态
- 查看个人信息、医疗记录、住院信息、费用明细
- 医保计算（60岁以上自费减半，演示用途）

### 医生端
- 查看排班和候诊队列
- 叫号就诊（自动呼叫下一位）
- 开立处方（药品库存同步）
- 申请检查（CT/B超/X光等）、检验（血常规/尿常规等）
- 开具住院通知
- 查看患者病史
- 结束看诊并生成诊断记录

### 护士端
- 分诊台：录入患者生命体征（体温、心率、血压等），收取分诊费
- 办理入院：处理医生开的住院申请，分配病房床位
- 每日护理记录录入（含护理费用）
- 办理出院结算（结算住院天数费用）

### 管理员端
- 科室与医生管理（增删改查）
- 全病区床位状态查询
- 药品库存管理：入库、添加新药、修改药品信息
- 药物消耗统计、患者统计、财务统计
- 打印全院统计报表
- 修改管理员密码

## 🛠 技术栈

| 方面 | 说明 |
|------|------|
| 语言 | C （C99 标准） |
| 数据结构 | 单向链表（各种业务实体） |
| 存储 | 二进制文件（`.dat`）持久化 |
| 界面 | Windows 控制台（`system("cls")` 清屏） |
| 编译环境 | MinGW-w64 / Visual Studio 均可 |

## 📂 目录结构

├── patient.c/h     # 患者模块
├── doctor.c/h      # 医生模块
